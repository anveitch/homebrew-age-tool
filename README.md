# homebrew-age-tool

Homebrew tap for [age-tool](https://github.com/anveitch/age-tool), a full featured AGE encryption management tool for encrypting, decrypting and managing AGE key pairs with audit logging.

## Installation

```bash
brew tap anveitch/age-tool
brew install age-tool
```

## Usage

Once installed, run:

```bash
age-tool
```

For full documentation and usage instructions, please see the main [age-tool repository](https://github.com/anveitch/age-tool).
